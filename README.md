# fgz

**fgz** is a mermaid-inspired, plain-text DSL + toolchain for factor graphs and Bayes nets, designed for robotics papers.

## Status

⚠️ **Early Development**: This repository is in early scaffold stage. The project is being set up and core functionality is under development.

### Near-term Plan
- DSL specification
- `fgz2tex` converter
- `fgz2svg` converter
- Editor tooling (separate repository)
