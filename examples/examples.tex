\section*{Progressive Examples}

Every fgz file starts with \texttt{fgz 1}. Positions are given in TikZ units, and the
generated snippets assume the paper preamble already includes \verb|\input{fgz.tikz.tex}|.

Start with two variables and a binary factor. Because the factor has no explicit
position, fgz places it halfway between the first two variables.

\subsection*{10. A Colored Chain Model}
This example combines direct LaTeX-style labels, square factors, colors, and higher-arity
factors in a temporal chain. The red factors are positioned explicitly, while the blue and
green ternary factors rely on midpoint placement along their first two variables. The
document-level style line nudges node size, factor size, label padding, and label font for
this one figure without changing the shared TikZ support file.
