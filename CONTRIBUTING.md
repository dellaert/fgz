# Contributing to fgz

Thank you for your interest in contributing to fgz!

## How to Contribute

### Reporting Issues

If you encounter a bug or have a feature request:
1. Check existing [issues](https://github.com/dellaert/fgz/issues) to avoid duplicates
2. Open a new issue with a clear description
3. Include relevant details (steps to reproduce, expected behavior, etc.)

### Submitting Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Run formatting and tests (once they exist)
5. Commit your changes with clear messages
6. Push to your fork
7. Open a Pull Request with a description of your changes
